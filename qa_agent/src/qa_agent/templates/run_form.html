Auto explore

Used when Auto explore is selected. Runs noop flow execution plus the auto_explore_ui plugin (Playwright). Password is kept in memory for the run only and is not written to meta.json.

Set Application (above) to load an optional profile from config/applications/<application>.yaml (e.g. coredge-console). The profile can supply base_url, login hints, navigation, safe_mode, and feature_keywords for selective runs.

If you enter a Target URL starting with http:// or https:// but leave Known flow selected, the server still runs Auto explore so Playwright is not skipped by mistake.
